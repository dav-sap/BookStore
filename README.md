# BookStore
This is a basic website for managing a book store.
To start the server, go to server folder and run 
```
npm install

npm start
```
## Dependecies
Make sure you have
```
1.node+npm
2.mongoDB, add mongod to you PATH Enviroment variables
```

server will be hosted on your localhost at:

```
http://localhost:3000/
```
